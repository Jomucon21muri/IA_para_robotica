# Visión por Computadora

Propósito:
- Trabajar en percepción visual: detección, segmentación, estimación de pose y reconstrucción 3D.

Contenido sugerido:
- `detection/` : detección de objetos y personas (YOLO, Faster-RCNN)
- `segmentation/` : segmentación semántica y por instancia
- `pose_estimation/` : estimación de pose 2D/3D, SLAM visual
- `datasets/` : scripts para convertir y probar datasets (COCO, KITTI, custom)

Cómo usar:
- Notebooks de ejemplos y checkpoints preentrenados para pruebas rápidas.
