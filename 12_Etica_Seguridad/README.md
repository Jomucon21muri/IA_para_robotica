# Ética, Seguridad y Buenas Prácticas

Propósito:
- Documentar consideraciones éticas, de seguridad y compliance para proyectos de IA en robótica.

Contenido sugerido:
- `risk_assessment/` : análisis de riesgos por módulo
- `safety_checklists/` : listas de verificación antes de pruebas en campo
- `ethics/` : principios, guías y referencias legales
- `incident_reports/` : plantilla para reportar y analizar incidentes

Cómo usar:
- Procedimientos mínimos obligatorios antes de ejecutar robots en entornos reales.
