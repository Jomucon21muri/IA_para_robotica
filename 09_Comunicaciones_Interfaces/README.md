# Comunicaciones e Interfaces (ROS, Middleware)

Propósito:
- Reunir patrones de integración: ROS (1/2), DDS, gRPC, APIs y formatos de mensajes.

Contenido sugerido:
- `ros/` : paquetes, mensajes personalizados y ejemplos de nodos
- `networking/` : topologías de red, QoS y sincronización entre módulos
- `api/` : especificaciones de APIs REST/gRPC para control remoto y telemetría
- `tools/` : utilidades para visualizar tópicos y mensajes

Cómo usar:
- Ejemplos de publicación/suscripción, creación de mensajes y conversiones.
