# Planificación (Path & Motion Planning)

Propósito:
- Reunir algoritmos de planificación de rutas y trayectorias: A*, D*, RRT, RRT*, PRM, trayectoria óptima.

Contenido sugerido:
- `global_planning/` : planificación de alto nivel en mapas
- `local_planning/` : controladores locales y evasión de obstáculos
- `trajectory_optim/` : optimización de trayectorias (CHOMP, TrajOpt)
- `simulations/` : escenarios de prueba y scripts para evaluar planners

Cómo usar:
- Integrar con módulos de percepción y control; incluir ejemplos con mapas y obstáculos dinámicos.
