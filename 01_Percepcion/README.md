# Percepción

Propósito:
- Reunir algoritmos y ejemplos para el procesamiento de datos de sensores: LIDAR, ultrasonidos, IMU, cámaras y sensores de proximidad.

Contenido sugerido:
- `sensors/` : descripciones y drivers (ej. `lidar`, `imu`, `cam`)
- `fusión/` : técnicas de sensor fusion (EKF, UKF, Kalman, filtros complementarios)
- `preprocesamiento/` : calibración, filtrado, normalización
- `notebooks/` : pruebas y visualizaciones

Cómo usar:
- Entradas: datos crudos de sensores.
- Salidas: representaciones filtradas y listas para localización/planificación.

Referencias rápidas:
- Papers y librerías recomendadas (ROS drivers, OpenCV, PCL).
