# Aprendizaje Automático y Aprendizaje por Refuerzo

Propósito:
- Explorar y aplicar técnicas de aprendizaje supervisado, no supervisado y por refuerzo a tareas robóticas.

Contenido sugerido:
- `supervised/` : modelos para percepción y control (CNNs, transformers ligeros)
- `rl/` : entornos, agentes, recompensas y experimentos (PPO, SAC, DDPG)
- `imitation/` : aprendizaje por imitación y análisis de demostraciones
- `datasets/` : enlaces y scripts de creación/conversión a formatos usados

Cómo usar:
- Scripts para entrenar, evaluar y desplegar modelos en simulación y hardware.
