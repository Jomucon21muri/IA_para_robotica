# Integración de Hardware

Propósito:
- Documentar la integración con microcontroladores, motores, controladores y electrónica del robot.

Contenido sugerido:
- `drivers/` : drivers específicos y flujos de comunicación (UART, SPI, I2C)
- `wiring/` : diagramas y guías de conexión
- `firmware/` : ejemplos o enlaces al firmware usado (STM, Arduino, ESP)
- `safety/` : procedimientos de seguridad para pruebas en hardware

Cómo usar:
- Guías paso a paso para montaje y conexión, checklist antes de pruebas físicas.
