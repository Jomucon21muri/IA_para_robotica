# Herramientas y Utilidades

Propósito:
- Almacenar herramientas de apoyo: scripts de despliegue, wrappers, utilidades de visualización y benchmarking.

Contenido sugerido:
- `scripts/` : utilidades para conversiones, despliegue y mantenimiento
- `viz/` : herramientas de visualización (RViz configs, matplotlib, dashboards)
- `ci/` : plantillas y ejemplos para integración continua
- `templates/` : plantillas para nuevos módulos o experimentos

Cómo usar:
- Instrucciones para añadir nuevas utilidades y buenas prácticas de mantenimiento.
