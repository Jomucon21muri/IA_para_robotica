# Simulación y Entornos de Prueba

Propósito:
- Centralizar simuladores y escenarios reproducibles para desarrollo (Gazebo, Webots, PyBullet, Coppelia).

Contenido sugerido:
- `launch/` : ejemplos de lanzamiento (launch files de ROS, scripts de inicio)
- `worlds/` : mundos y escenas de prueba
- `benchmarks/` : scripts para ejecutar pruebas automatizadas y comparar resultados
- `ci/` : integración y validación (opcional)

Cómo usar:
- Instrucciones para ejecutar escenarios, grabar `rosbag` y reproducir experimentos.
